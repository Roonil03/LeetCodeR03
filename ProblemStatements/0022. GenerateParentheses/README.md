# 22. Generate Parentheses
## Question Level: Medium
### Description:
Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

### Examples:
<b>Example 1:</b><br>
Input: n = 3<br>
Output: ``["((()))","(()())","(())()","()(())","()()()"]``<br>

<b>Example 2:</b><br>
Input: n = 1<br>
Output: ``["()"]``<br>

### Constraints:

- 1 <= n <= 8

### <i>Concepts Used:
- String
- Dynamic Programming
- Backtracking </i>
