# 32. Longest Valid Parentheses
## Question Level: Hard
### Description:
Given a string containing just the characters '(' and ')', return the length of the longest valid (well-formed) parentheses 
substring.

### Examples:
#### Example 1:

Input: s = "``(()``"<br>
Output: 2<br>
Explanation: The longest valid parentheses substring is "``()``".<br>
#### Example 2:

Input: s = "``)()())``"<br>
Output: 4<br>
Explanation: The longest valid parentheses substring is "``()()``".<br>
#### Example 3:

Input: s = ""<br>
Output: 0<br>

### Constraints:

- 0 <= ``s.length`` <= 3 * 10^4
- s[i] is '``(``', or '``)``'.

### <i>Concepts Used:
- String
- Dynamic Programming
- Stack </i>
