# 5. Longest Palindromic Substring
## Question Level: Medium
### Description:
Given a string s, return the longest palindromic substring in s.

### Example:
Example 1:<br>
Input: s = "babad"<br>
Output: "bab"<br>
Explanation: "aba" is also a valid answer.<br>

Example 2:<br>
Input: s = "cbbd"<br>
Output: "bb"<br>

### Constraints:
1 <= s.length <= 1000
s consist of only digits and English letters.
